# 🃏 Flashcards

A simple flashcard study app that runs in the browser. Create decks, add cards, study by flipping them.

My second coding project — built ticket-by-ticket from a design, using an agent pipeline.

## How to run it
Open `index.html` in any web browser.

## Built with
- HTML (structure)
- CSS (styling)
- JavaScript (behavior)
- Data saved in the browser via localStorage
